# QuadTodo 详细设计方案

> 基于 Qt 6 QML 的四象限时间管理桌面应用

## 1. 项目概述

### 1.1 产品定义

**QuadTodo** 是一款基于四象限时间管理法（艾森豪威尔矩阵）的桌面应用，采用 Qt 6 QML 技术栈构建，使用 SQLite 本地存储，完全离线运行，支持中英文双语。

### 1.2 技术栈

| 维度 | 选型 |
|------|------|
| 语言 | C++17 / QML/JavaScript |
| UI 框架 | Qt 6 Quick (QML) |
| 渲染引擎 | Qt Scene Graph (GPU) |
| 数据绑定 | QML Property Bindings |
| 样式系统 | QML 声明式主题 + 属性绑定 |
| 布局系统 | QML Layout / SplitView |
| 列表渲染 | QML ListView + Delegate |
| 数据模型 | C++ QAbstractListModel + QSortFilterProxyModel |
| 数据库 | SQLite (QSqlDatabase) |
| 配置管理 | QSettings (IniFormat) |
| 国际化 | Qt6 qt_add_translations |
| 构建系统 | CMake + qt_add_qml_module |
| 安装打包 | Inno Setup |

### 1.3 功能清单

| # | 功能 | 说明 |
|---|------|------|
| 1 | 四象限任务视图 | 2×2 网格布局，任务按紧急/重要维度分布 |
| 2 | 多时间维度视图 | 正常/昨日/今日/本周/上周/本月/上月/本季/上季/半年/年度，共 13 种视图 |
| 3 | 任务增删改查 | 新建、编辑、删除、搜索任务 |
| 4 | 任务完成/恢复 | 勾选完成，支持恢复 |
| 5 | 软删除与回收站 | 删除进入回收站，支持恢复/彻底删除/清空 |
| 6 | 任务置顶 | 置顶/取消置顶，置顶任务排在列表顶部 |
| 7 | 任务移动象限 | 将任务从一个象限移动到另一个象限 |
| 8 | 任务复制 | 复制内容到剪贴板 / 复制为新任务 |
| 9 | 重复任务 | 每日/每周/每月/每年/自定义，到期自动重置 |
| 10 | 提醒功能 | 设置提醒时间，到时弹窗通知 |
| 11 | 截止日期 | 设置截止时间，过期标记高亮 |
| 12 | 任务链接 | 为任务关联 URL，一键打开 |
| 13 | 富文本备注 | 加粗/斜体/下划线/字号/图片嵌入 |
| 14 | 任务标签 | 为任务设置标签，按标签搜索 |
| 15 | 任务搜索 | 全文搜索：描述/时间/地点/人物/链接/备注 |
| 16 | 多种排序 | 按描述/时间/地点/人物/标签/创建时间/完成状态/提醒/截止/重复/删除 排序 |
| 17 | 完成淡出动画 | 任务完成后渐隐消失 |
| 18 | 系统托盘 | 最小化到托盘，托盘右键菜单 |
| 19 | 主题系统 | 深色/浅色/跟随系统，全局实时切换 |
| 20 | 字体大小调节 | 8 级字体大小，UI 自适应 |
| 21 | 置顶提示模式 | 迷你窗口常驻桌面，可调比例和透明度 |
| 22 | 自动备份 | 定时备份数据库，保留最近 5 份 |
| 23 | 象限顺序自定义 | 拖拽调整四个象限在网格中的位置 |
| 24 | 全局快捷键 | Ctrl+F 搜索 / Ctrl+R 切换完成 / Ctrl+D 切换删除 / Ctrl+Tab 切换视图 |
| 25 | 单实例限制 | 同一时间只允许运行一个实例 |
| 26 | Windows 深色标题栏 | 通过 DWM API 设置标题栏深色模式 |
| 27 | 国际化 | 中文(zh_CN) / 英文(en_US) |
| 28 | 新手引导 | 首次启动创建示例任务 |
| 29 | 关于对话框 | 显示版本号和应用信息 |
| 30 | 右键上下文菜单 | 任务项右键菜单（置顶/完成/提醒/截止/重复/移动/删除等） |
| 31 | 通知弹窗 | 提醒/视图/搜索三种通知类型，顶部居中显示 |
