## 1. 动态规划（DP）

动态规划适用于这两类特征明显的问题：

- **重叠子问题**：同一个子问题会被反复计算  
- **最优子结构**：原问题最优解可以由子问题最优解组合得到

核心是找到：**状态定义 + 状态转移方程**。  
一旦状态转移方程写出来，问题基本就解决了。

### 两种实现方式

1. **自顶向下（递归 + 备忘录）**
   - 先写递归（穷举决策树）
   - 用 memo 记录结果，避免重复计算（剪枝）

2. **自底向上（DP Table）**
   - 按状态依赖顺序填表
   - 通常时间稳定，便于优化空间

---

### 例子：0/1 背包

> 有 `n` 件物品，每件有重量 `w[i]`、价值 `v[i]`，背包容量 `W`，每件最多选一次，求最大价值。

状态定义：`dp[i][j]` 表示前 `i` 件物品，容量为 `j` 时的最大价值。  
转移：
- 不选第 `i` 件：`dp[i-1][j]`
- 选第 `i` 件（若 `j>=w[i]`）：`dp[i-1][j-w[i]] + v[i]`

```cpp
#include <bits/stdc++.h>
using namespace std;

int knapsack01(vector<int>& w, vector<int>& v, int W) {
    int n = w.size();
    vector<vector<int>> dp(n + 1, vector<int>(W + 1, 0));

    for (int i = 1; i <= n; ++i) {
        for (int j = 0; j <= W; ++j) {
            dp[i][j] = dp[i - 1][j]; // 不选
            if (j >= w[i - 1]) {
                dp[i][j] = max(dp[i][j], dp[i - 1][j - w[i - 1]] + v[i - 1]); // 选
            }
        }
    }
    return dp[n][W];
}
```

## 2. 贪心算法

贪心思想：**每一步都做当前看起来最优的选择**。  
关键在于证明“局部最优能导向全局最优”。

### 例子：活动选择问题

> 给定若干活动开始/结束时间，选最多个互不冲突活动。

策略：按结束时间升序排序，优先选结束最早的活动。

```cpp
#include <bits/stdc++.h>
using namespace std;

int maxActivities(vector<pair<int,int>>& acts) {
    sort(acts.begin(), acts.end(), [](auto& a, auto& b){
        return a.second < b.second; // 按结束时间排序
    });
    int cnt = 0, lastEnd = INT_MIN;
    for (auto &a : acts) {
        if (a.first >= lastEnd) {
            ++cnt;
            lastEnd = a.second;
        }
    }
    return cnt;
}
```

## 3. 回溯算法

回溯本质：**决策树遍历**。  
你只要想清楚三件事：

1. **路径**：已经做出的选择
2. **选择列表**：当前可选项
3. **结束条件**：何时收集结果/停止

通用框架：

```cpp
void backtrack(路径, 选择列表) {
    if (结束条件) {
        记录结果;
        return;
    }
    for (选择 in 选择列表) {
        做选择;
        backtrack(路径, 新选择列表);
        撤销选择;
    }
}
```

---

### 示例1：全排列

```cpp
#include <bits/stdc++.h>
using namespace std;

vector<vector<int>> res;
vector<int> path;
vector<bool> used;

void dfs(vector<int>& nums) {
    if (path.size() == nums.size()) {
        res.push_back(path);
        return;
    }
    for (int i = 0; i < nums.size(); ++i) {
        if (used[i]) continue;
        used[i] = true;
        path.push_back(nums[i]);
        dfs(nums);
        path.pop_back();
        used[i] = false;
    }
}
```

### 示例2：八皇后（思路）
- 路径：前 `row` 行皇后放置位置
- 选择列表：当前行可放列
- 结束条件：`row == n`
- 剪枝：列、主对角线、副对角线不能冲突

## 4. 图的遍历与搜索

图常见表示：

- **邻接矩阵**：`O(V^2)` 空间，适合稠密图
- **邻接表**：`O(V+E)` 空间，适合稀疏图（更常用）

### DFS（有向图，邻接矩阵）

```cpp
#include <bits/stdc++.h>
using namespace std;

void dfsMatrix(int u, vector<vector<int>>& g, vector<bool>& vis) {
    vis[u] = true;
    for (int v = 0; v < g.size(); ++v) {
        if (g[u][v] && !vis[v]) dfsMatrix(v, g, vis);
    }
}
```

### BFS（无向图，邻接表）

```cpp
#include <bits/stdc++.h>
using namespace std;

void bfsList(int s, vector<vector<int>>& g) {
    vector<bool> vis(g.size(), false);
    queue<int> q;
    q.push(s); vis[s] = true;
    while (!q.empty()) {
        int u = q.front(); q.pop();
        for (int v : g[u]) {
            if (!vis[v]) {
                vis[v] = true;
                q.push(v);
            }
        }
    }
}
```

## 5. 链表边界条件检查清单

写链表题前，先过一遍这 5 条：

- 链表为空时是否正常？
- 只有 1 个结点时是否正常？
- 只有 2 个结点时是否正常？
- 头结点处理是否正确？
- 尾结点处理是否正确（尤其 `next == nullptr`）？

这一步能避免 80% 的链表 bug。

## 6. 双指针常见应用（链表）

经典套路：

1. **判断是否有环**：快慢指针（Floyd）
2. **找环入口**：相遇后一个回头同步走
3. **找中点**：快慢指针（归并排序常用）
4. **找倒数第 k 个**：先让 fast 走 k 步，再双指针同步

## 7. 区间问题

区间题通用思路：

1. 先排序（按起点或终点）
2. 再线性扫描找规律（合并、选择、计数）

常见题型：  
- 合并区间  
- 不重叠区间个数  
- 最少删除区间  
- 会议室安排

## 8. 常用位操作速记

```cpp
x & 1           // 判断奇偶
x >> 1          // 除以2（整数）
x << 1          // 乘以2
x & (x - 1)     // 去掉最低位的1
x & -x          // 取最低位的1（lowbit）
x | (1 << k)    // 将第k位置1
x & ~(1 << k)   // 将第k位置0
x ^ (1 << k)    // 翻转第k位
```

位运算常用于：状态压缩 DP、子集枚举、快速判重、二进制技巧题。

## 结语
- **动态规划**：明确状态和转移  
- **贪心**：证明局部最优可达全局最优  
- **回溯**：在决策树上搜索 + 剪枝  
- **图搜索**：DFS/BFS 模板化  
- **链表/双指针**：边界优先  
- **区间/位运算**：高频套路，熟能生巧
