在 GitHub 上为现有仓库添加开源许可证（License）非常方便，GitHub 提供了内置的模板工具。

## 核心步骤

- 进入仓库：在 GitHub 上打开你的目标仓库。
- 创建新文件：点击文件列表上方的 Add file 按钮，选择 Create new file。
- 命名文件：在文件名输入框中输入 LICENSE（必须全大写）。

使用模板：
  
- 一旦你输入了文件名，右侧会出现 Choose a license template 按钮，点击它。
- 在左侧列表中选择适合你的协议（如 MIT, Apache 2.0, GPL 等）。
- 在右侧预览区填写 Year（年份）和 Full name（你的名字或组织名）。
- 提交更改：点击 Review and submit，然后点击页面底部的 Commit changes 按钮完成添加。
  
## 如何选择合适的许可证？

如果你不确定该选哪一个，可以参考 GitHub 官方推荐工具 choosealicense.com： 

- MIT：最简单宽松，只要保留原作者信息，别人可以随意使用和商用。
- Apache 2.0：类似于 MIT，但额外增加了明确的专利授权保护。
- GPL v3：最严格，要求任何基于此代码开发的衍生软件也必须开源。
