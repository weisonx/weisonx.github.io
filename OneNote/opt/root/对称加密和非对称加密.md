# 对称加密
对称加密和非对称加密都是加密算法，用于保护数据的安全性。

对称加密是指加密和解密使用相同的密钥，也就是说，发送方和接收方都需要知道同一个密钥才能进行加密和解密。

对称加密算法的优点是加密和解密速度快，但缺点是密钥需要在发送方和接收方之间共享，如果密钥被泄露，那么加密的数据就会被攻击者轻易地解密。

# 非对称加密

非对称加密是指加密和解密使用不同的密钥，也就是说，发送方使用接收方的公钥进行加密，接收方使用自己的私钥进行解密。

非对称加密算法的优点是密钥不需要共享，因此更加安全，但缺点是加密和解密速度较慢。

# 应用
通常情况下，对称加密算法用于加密大量数据，而非对称加密算法用于加密少量数据或者用于数字签名等场景。
