# 可视化UI设计 `QtCreator`

* ui文件被转换为C++类定义文件的过程和使用原理
* 创建项目、配置文件
* 双击ui文件，进行可视化设计UII（包括组件属性设置；连接组件的公有信号、公有槽）
* 构建项目，生成`ui_`类文件，定义了ui里面的组件

与窗口相关的4个文件
|文件| 说明|
| -- | -- |
| widget.h| 定义了窗口类Widget |
| widget.cpp | 实现了Widget类的功能的源程序文件 |
| widget.ui | 窗口UI文件，用于在QtDesigner中进行UI可视化设计。wiget.ui是一个XML文件，存储界面上各个组件的属性和布局内容 |
| ui_wiget.h | UI文件经过UIC编译后生成的文件，这个文件定义了一个类，类的名称是Ui_Widget，用C++语言描述UI文件中界面组件的属性设置、布局以及信号与槽的关联等内容|

* `widget`类创建窗口并初始化、访问ui_类里的组件
* 主程序执行窗口程序
